# AI Cora Duplicate

This is a duplicate of the AI Cora dashboard website (https://aicora.aisuites.io/dashboard).

## Structure

- `index.html` - Main entry point (React SPA)
- `assets/` - Static assets (JS, CSS, images)
- `logo-sm.png` - Site logo
- `favicon.ico` - Favicon

## Pages/Routes Identified

The following routes are handled by the React Router in this SPA:

- `/` - Home
- `/login` - Login
- `/dashboard` - Dashboard
- `/agents` - Agents
- `/clients` - Clients
- `/chats` - Chats
- `/content` - Content
- `/contents` - Contents
- `/editor` - Editor
- `/image-editor` - Image Editor
- `/ideation-generator` - Ideation Generator
- `/integrations` - Integrations
- `/my-account` - My Account
- `/account-management` - Account Management
- `/agency-management` - Agency Management
- `/schedule` - Schedule
- `/news` - News
- `/media` - Media
- `/upgrades` - Upgrades
- `/help-support` - Help Support
- `/help-post` - Help Post
- `/support-article` - Support Article
- `/privacy` - Privacy
- `/forgot-password` - Forgot Password
- `/reset-password` - Reset Password
- `/master-login` - Master Login
- `/newbot` - New Bot
- `/me` - Me
- `/tiktok-callback` - TikTok Callback
- `/twitter-callback` - Twitter Callback
- `/whatsapp-connect` - WhatsApp Connect

## Deployment

This site is configured for deployment on Vercel.

## Note

This is a static duplicate of a React Single Page Application (SPA). The original site requires authentication for most functionality.
